CALDERA

Cyber Adversary Language and Decision Engine for Red Team Automation

CALDERA is a framework designed to quickly run breach-and-simulation exercises. It was originally developed as a MITRE research & development program, aimed at automating the decision-making of a cyber adversary. Nowadays, it can also be used in realistic red-and-blue exercises. CALDERA is built on top of the ATT&CK framework.


1

Deploy agents

Deploy Sandcat (54ndc47) agents on each compromised computer.

2

Build adversaries

Combine ATT&CK TTPs to build custom adversaries, representing realistic threats.

3

Run operations

Point your adversary against the computers running agents to start an operation.
